# VHALVersion

`VHALVersion` struct provides compile-time version information for the VHAL library.

Header: `#include <VHALVersion.h>`

## Current Version

```cpp
auto version = VHALVersion::Current();  // { 0, 1, 0 }
const char* str = VHALVersion::String(); // "0.1.0"
```

## API

| Method / Field | Type | Description |
|----------------|------|-------------|
| `major` | `uint8` | Major version number |
| `minor` | `uint8` | Minor version number |
| `patch` | `uint16` | Patch version number |
| `Current()` | `static constexpr VHALVersion` | Returns the current VHAL version |
| `String()` | `static constexpr const char*` | Returns the version as a string (`"0.1.0"`) |
| `UInt32()` | `constexpr uint32` | Packs version into a single 32-bit integer: `(major << 24) | (minor << 16) | patch`. Useful for numeric comparisons |

## Comparison Operators

All comparison operators are `constexpr` and work on the packed `UInt32()` representation:

```cpp
VHALVersion required = { 0, 2, 0 };
if (VHALVersion::Current() < required) {
    // VHAL version is too old
}
```

Supported: `==`, `!=`, `<`, `<=`, `>`, `>=`.

## Use Cases

- **Firmware version reporting** — embed in telemetry or UART output
- **Compatibility checks** — verify VHAL version at compile time or boot
- **Bootloader metadata** — store alongside firmware image for update verification
