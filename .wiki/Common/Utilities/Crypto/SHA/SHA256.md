# SHA256

Header-only SHA-256 cryptographic hash function producing a 256-bit (32-byte) digest.

## Quick Start

```cpp
#include <Utilities/Crypto/SHA/SHA256.h>

SHA256 hasher;
hasher.Update("Hello World");
std::array<uint8, 32> hash = hasher.Finalize();
```

## Features

### Hashing a String

`Update` accepts `std::string_view`, so string literals, `std::string`, and views all work directly.

```cpp
SHA256 hasher;
hasher.Update("password123");
auto hash = hasher.Finalize(); // std::array<uint8, 32>
```

### Hashing Binary Data

`Update` also accepts `std::span<const uint8>` for raw byte buffers.

```cpp
std::array<uint8, 1000> data = { /* ... */ };

SHA256 hasher;
hasher.Update(std::span(data));
auto hash = hasher.Finalize();
```

### Incremental Updates

Call `Update` multiple times to feed data in chunks. The final digest is the same as if all data were provided at once. This is useful for streaming or file integrity scenarios.

```cpp
SHA256 hasher;
hasher.Update("Part 1");
hasher.Update("Part 2");
hasher.Update("Part 3");
auto hash = hasher.Finalize();
```

File hashing example:

```cpp
SHA256 hasher;
while (readFileChunk(buffer)) {
    hasher.Update(std::span(buffer));
}
auto fileHash = hasher.Finalize();
```

### Reset and Reuse

After calling `Finalize`, call `Reset` to reuse the same instance for a new hash computation.

```cpp
SHA256 hasher;

hasher.Update("First message");
auto hash1 = hasher.Finalize();

hasher.Reset();

hasher.Update("Second message");
auto hash2 = hasher.Finalize();
```

## API Reference

### Class `SHA256`

| Method | Signature | Description |
|---|---|---|
| **Constructor** | `SHA256()` | Initializes the hasher (calls `Reset` internally). |
| **Reset** | `void Reset()` | Resets internal state so the instance can be reused for a new hash. |
| **Update** | `void Update(std::span<const uint8> data)` | Feeds raw bytes into the hash computation. May be called multiple times. |
| **Update** | `void Update(std::string_view str)` | Feeds a string into the hash computation. Convenience overload that delegates to the span variant. |
| **Finalize** | `std::array<uint8, 32> Finalize()` | Applies SHA-256 padding, performs the final transform, and returns the 32-byte digest. Call `Reset` before reusing the instance after this. |

### Private Helpers

These implement the SHA-256 algorithm internals and are not part of the public interface:

| Function | Purpose |
|---|---|
| `Rotr(x, n)` | 32-bit right rotation. |
| `Ch(x, y, z)` | Choice function. |
| `Maj(x, y, z)` | Majority function. |
| `Sigma0(x)` | Upper-case sigma-0. |
| `Sigma1(x)` | Upper-case sigma-1. |
| `Gamma0(x)` | Lower-case sigma-0 (message schedule). |
| `Gamma1(x)` | Lower-case sigma-1 (message schedule). |
| `Transform()` | Processes one 64-byte block and updates the state. |
